HTML5

HTML or Hyper-Text Markup Language is the primary language used in the creation of web pages. Since its creation in 1993 by Tim Berners-Lee, one of the original creators of the internet, it has been a backbone of web pages around the world.

Over the years it has undergone a many updates and revisions. The most recent version, HTML5, was released in 2008 and introduced a slew of new features aimed primarily at supporting new forms of multimedia and dynamic graphical content.


The Canvas Tag

The Canvas tag can be used in combination with JavaScript to draw bitmap images to the screen.




function drawRect () {
  aCanvas = document.getElementById("canvas-one");
  aContext = aCanvas.getContext("2d");
  aContext.fillStyle = "black";
  aContext.fillRect(50, 60, 200, 120);
}

<button onclick="drawRect()">Default</button>
<canvas id="canvas-one" height="240" width="300"></canvas>

SVG

SVG or Scalable Vector Graphics are used to draw vector graphics to the screen which can be rescaled to any size without pixelating. Another advantage of SVG over the Canvas tag is that because they accessible via the DOM, JavaScript event handlers such as click events can be assigned to them. Click the image below to change its colour.


function svgClick () {
  var circ = document.getElementById("circle");
  if (circ.getAttribute("fill") == "#C7C9BE") {
    circ.setAttribute("fill", "#89CEDE");
  } else {
    circ.setAttribute("fill", "#C7C9BE");
  }
}

<svg id="svg" height="240" width="300">
  <circle id="circle" cx="150" cy="120" r="100" fill="#C7C9BE" stroke="#DC4E00" stroke-width="2" onclick="svgClick()"></circle>
</svg>
