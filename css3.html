CSS3

CSS or Cascading Style Sheets are used to describe the formatting and style of a web page written in a markup language. Prior to the use of CSS, formatting was done using HTML which often led to messy code which was difficult to manage. Like HTML, there have been several versions of CSS since its inception in 1996. The current version, CSS3, contains many new features aimed at creating more dynamic and interesting web content.



Gradients

Prior to CSS3, pre-rendered images were required in order to display gradients but with CSS3, gradients can be created using stylesheets.


#gradient {
  display: block;
  height: 200px;
  width: 300px;
  border: 1px solid #929487;
  background: -webkit-linear-gradient(#DC4E00, #89CEDE);
  background: -o-linear-gradient(#DC4E00, #89CEDE);
  background: -moz-linear-gradient(#DC4E00, #929487);
  background: linear-gradient(#89CEDE, white);
}

<div id="gradient"></div>


Custom Fonts

CSS3 allows web designers to make use of custom fonts without having to worry about whether the end-user has the font installed on their machine or not.

This is a custom font.

@font-face {
  font-family: signaritaChloe;
  src: url(fonts/signarita_chloe.ttf)
}


Animations

Using CSS3, designers can easily animate changes in formatting.


#animation {
  height: 200px;
  width: 200px;
  animation: frames 5s infinite;
}

@keyframes frames {
  0% {background: #89CEDE;}
  25% {margin-left: 50px;}
  50% {width: 300px;}
  100% {background: white;}
}
