JavaScript

JavaScript is a primarily client-side scripting language used commonly throughout the web to create dynamic web pages in a way not possible with HTML or CSS alone. Common uses of JavaScript include: animating page elements, embedding audio or video into a web page, validating web forms, and detecting user input such as keystrokes. Although JavaScript shares similar syntax and naming conventions with Java, it is an entirely different language with different semantics.



Sequential Programming

This is an example of sequential programming using JavaScript.


function jsSequence () {
  alert("Step One");
  alert("Step Two");
  alert("Step Three");
}

<button onclick="jsSequence()>JS Sequence</button>








If Statement

This is a conditional If statement which gives a different output depending on the user's input.




function jsIf() {
if (isNaN(prompt("Enter a word or number: ")))
  alert("You entered a word");
else
  alert("You entered a number");
}

<button onclick="jsIf()">JS If</button>


While Loop




This is a while loop which repeats the same action until a specified condition is met.


function jsWhile() {
var counter = 1;
  while (counter < 4) {
    alert("Step " + counter);
    counter++;
  }
}

<button onclick="jsWhile()">JS While</button>
